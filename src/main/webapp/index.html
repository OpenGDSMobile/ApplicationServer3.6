OpenGDS Mobile

Hansung University in South Korea kilee@hansung.ac.kr


OpenGDS Mobile 성과물인 MoWMaS(Mobile Web Mapping Solution)은 모바일 웹에서 공공 데이터와 공간정보를
연계하여 시각화할 수 있는 서버 및 클라이언트 모듈이다.
여러 시각화 및 공간정보 관련 오픈소스가 활용되었으며, 이를 기반 또는 API를 활용하여 추가 개발하여 관련된 서비스를 빠르고 쉽게 제공할 수 있다.
MoWMaS는 국토교통부 국토공간정보연구사업 공간정보 SW활용을 위한 오픈소스 가공기술 개발
(모바일 공간정보 서비스 기술개발, No. 14NSIP-B080144-01)의 연구 성과를 일부 포함하고 있습니다.
⒞2018. 한성대학교 산학협력단 공간정보 웹 기술 연구실

Modules

Application Server

Based on eGovernment Environment
  • 전자정부 공통컴포넌트 연동
  • RESTful API 기반 공공데이터 및 공간정보 요청
  • 서울열린데이터광장 데이터 스케줄러 기반 요청 예제 소스코드 및 데모 제공
  • 오픈소스 공간정보 서버(GeoServer) 활용 공간 데이터 활용 인터페이스 제공

Client

Based on eGovernment Mobile Environment
  • 전자정부 모바일 실행환경 컴포넌트 제공
  • OpenLayers 3 활용 패키징 공간정보 데이터 시각화 편의 제공(OpenGDS Mobile Client Library)
  • 공간 데이터 시각화 레이어 관리 UI 제공
  • D3.js 활용 기본 차트 시각화 편의 제공

Linker

Based on Shell Script
  • Application Server 및 Mobile Client 개발 및 운영환경 구성
  • Ubuntu / CentOS / Windows 환경 제공
