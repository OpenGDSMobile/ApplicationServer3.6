OpenGDS Mobile Module

서버 클라이언트 활용 오픈소스 라이선스

소 개

OpenGDS Mobile Module은 모바일 웹에서 공공 데이터와 공간정보를 연계하여 시각화할 수 있는 서버 및 클라이언트이다.
여러 시각화 및 공간정보 관련 오픈소스가 활용되었으며, 이를 기반 또는 API를 활용하여 추가 개발하여 관련된 서비스를 빠르고 쉽게 제공할 수 있다.

참고 MoWMaS는 국토교통부 국토공간정보연구사업 공간정보 SW활용을 위한 오픈소스 가공기술 개발
(모바일 공간정보 서비스 기술개발, No. 14NSIP-B080144-01)의 연구 성과를 일부 포함하고 있습니다.

MoWMaS Application Server

전자정부 모바일 표준프레임워크 프로젝트 기반으로 개발된 자바 기반의 서버

기능
  • 전자정부 공통컴포넌트 연동 가능
  • RESTful API 공공데이터 / 공간정보 요청
  • 공공데이터 포털(http://data.go.kr) 데이터 요청
  • 서울열린데이터광장(http://data.seoul.go.kr) 데이터 요청
  • 공간정보 서버(GeoServer) 관리 인터페이스 제공
공공 데이터 수집 관리 페이지 단위 예제 바로가기

MoWMaS Mobile Client

전자정부 모바일 표준프레임워크 실행환경 기반으로 개발된 자바스크립트 라이브러리

단위 예제 바로가기

기능
  • 전자정부 모바일 실행환경 컴포넌트 UI/UX 편리 생성 기능
  • OpenLayers 3 활용 공간정보 데이터 시각화
  • 공간정보 시각화 레이어 관리 기능
  • WebSocket 활용 속성정보 온/오프라인 편집 기능
  • D3.js 활용 차트 시각화

활용 오픈소스

모바일 웹 매핑 솔루션은 공간정보 및 시각화 관련 라이브러리가 다수 사용되었다. 또한 서비스를 하기 위해 사용되는 소프트웨어가 존재한다. 이는 모두 오픈소스로 구성되어 있으며, 목록 및 라이선스는 다음과 같다.

오픈소스 목록

  • MoWMaS Application Server
    1. 전자정부 표준프레임워크 (http://www.egovframe.go.kr)
      : java기반 정보시스템 구축 시 개발 및 운영 표준환경 제공
    2. Spring Framework (https://projects.spring.io)
      : 자바 어플리케이션 웹 프레임워크
    3. MyBatis (http://www.mybatis.org)
      : 관계형 테이터베이스 프레임워크
      : SQL 데이터베이스와 자바 연동 프레임워크
    4. Jackson
      : 다양한 형식의 데이터를 지원하는 data-processing 툴
    5. GeoSolutions (http://www.geo-solutions.it)
      : GeoServer를 다룰수 있도록 해주는 툴
    6. MojoHaus (http://www.mojohaus.org)
      : JAX-WS를 쉽게 해주는 메이븐을 위한 라이브러리
  • MoWMaS Mobile Client
    1. 표준프레임워크 모바일 실행환경
      : 모바일 화면처리에 최적화 된 UX제공
      • jQuery (https://jquery.com)
        : 자바스크립트 개발 라이브러리
      • jQuery Mobile (https://jquerymobile.com)
        : 모바일 웹 어플리케이션 개발을 위한 자바스크립트 라이브러리
    2. PROJ.4 (http://proj4.org)
      : 좌표계 변환시 사용
    3. OpenLayers 3 (http://openlayers.org)
      : 웹 매핑 자바스크립트 라이브러리
    4. D3(Data-Driven Documents) (http://d3js.org)
      : 데이터 시각화 자바스크립트 라이브러리
    5. DataTables (https://datatables.net/)
      : 테이블 시각화를 위한 jQuery 플러그인
    6. Sortable
      : 테이블 정렬시 사용
  • Software
    1. Apache (http://www.apache.org)
      : HTTP 웹 서버
    2. Tomcat (http://tomcat.apache.org)
      : 자바환경 웹 컨테이너 서버
    3. GeoServer (http://geoserver.org)
      : 웹 상에서 공간정보 관리 할 수 있는 자바 기반 소프트웨어
    4. PostgreSQL / PostGIS
      (https://www.postgresql.org / http://www.postgis.net/)
      : 공간정보를 지원하는 관계형 데이터베이스
